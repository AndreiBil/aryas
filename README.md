# Aryas-Bot
A bot for the Developers Discord server

## Running the bot
1 Clone the repo and `cd` into its root folder.

2. `$ docker build . -t aryas_bot`

3. `$ ./run.sh`

4. `$ docker rm -f [CONTAINER HASH]`

5. `Edit `~/.aryas/cfg.json`, add the necessary API keys.`

6. `$ ./run.sh`

7) ...

8) Profit!
